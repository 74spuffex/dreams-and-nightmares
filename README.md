1. Install the plugin by dropping the .jar file inside your servers plugins folder and restarting.
2. Enable the Skript via /sk enable nightmaresndreams.sk
3. Install 2 Dream & Nightmare Worlds of your choice and use their multiverse name in the options setting (Go Inside your plugins folder, then Skript folder, then scripts, and then click the skript and put in the names of the worlds). 
5. You're finished and the plugin should be working.

Ensure Skbee and Multi-Verse Core is enabled and installed.
